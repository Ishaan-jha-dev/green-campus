# GreenPulse Campus Ecosystem

GreenPulse is the institutional standard for campus sustainability intelligence. Built for top-tier Indian universities, it provides a high-fidelity platform for tracking carbon, auditing energy waste, and ensuring waste compliance through visual AI.

## 🚀 Intelligent Modules

- **Audit Map**: Real-time spatial visualization of campus energy load and efficiency nodes.
- **Energy Radar**: Institutional-grade utility auditing engine for identifying capital leakage.
- **Sustainability AI**: Generative intelligence for personalized campus decarbonization roadmaps.
- **Carbon Ledger**: Deep Scope 1, 2, and 3 emission tracking with automated reporting.
- **Waste Intelligence**: On-device neural processing for smart waste classification and EPR compliance.

## 🛠️ Tech Stack

- **Frontend**: Next.js 15+ (App Router), Tailwind CSS, Framer Motion
- **Intelligence**: TensorFlow.js, Generative AI Models
- **Compliance**: Verified carbon accounting frameworks

## 🏁 Deployment

```bash
npm install
npm run dev
```

## 🛡️ Institutional Integrity

GreenPulse is a privacy-first platform. All visual processing for waste classification occurs locally in-browser. Reports are cryptographically signed for immutable audit trails.
